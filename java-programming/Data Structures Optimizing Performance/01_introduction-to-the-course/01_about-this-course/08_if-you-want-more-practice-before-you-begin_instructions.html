Resources to help you prepare for this course

If you feel that you may not be ready for this course, here are some resources you can use to help you prepare, then come back when you feel ready. If you are ready, go ahead and skip this reading and continue with the course.

If you feel that you are just missing a little bit of background , try the first course in this specialization:

  • Object Oriented Programming in Java

If you need a more basic introduction to programming in Java , we recommend Duke's specialization: Java Programming: An Introduction to Software

If you know programming but are new to Java , you might find the Java tutorialspoint site or the Oracle Java tutorials useful to help you transition to Java.

There are countless other "learn Java" resources out there, and we hope that you'll come back once you feel comfortable and ready to go with this course.
