Is this course right for me?

This course is designated as Intermediate Level, with "some experience required." But you're probably wondering just what experience, exactly, is required. In this reading and the quiz that follows, we will try to help you determine whether or not you have the correct background for this course and this specialization.

We assume that you have taken and passed a college-level introductory Object Oriented Java programming course designed for CS majors , or its online equivalent. Topics in these courses typically include at least the following topics:

  • Variables and types

  • Basic data types: Strings, ints, doubles, etc.

  • Loops (for, while, nested loops)

  • Calling methods

  • Writing methods (with and without return types, with and without parameters)

  • Using objects and possibly designing them

  • 1D and 2D arrays

  • Basic input and output (System.out.println)

  • Compiling and running Java code

  • Creating and designing classes

  • Variable Scope

  • Inheritance and Polymorphism

  • Abstract classes and Interfaces

  • Searching arrays

One example of such a course is UC San Diego's CSE 11. If the content in the course schedule (bottom of that page) looks familiar to you, you're in good shape. Here at Coursera, we've designed this course to follow the course: " Object Oriented Programming in Java ". If you have some background in Java but are unfamiliar with some of the topics above (like Inheritance), we recommend you start with that course.

If you are brand new to programming, you may want to start with the introductory Java specialization " Java Programming: An Introduction to Software " offered by Duke University.

What if I have taken an object oriented introductory programming course, but it wasn't in Java?

If you have taken a course that covers all or most of the above topics, but it was in a language like C, C++, or python, you will probably do fine in our specialization. We recommend that you review Java syntax and some details of how Java works with objects using some basic online resources (e.g. http://www.tutorialspoint.com/java/java_basic_syntax.htm ). You may benefit from reviewing information in our prior course: " Object Oriented Programming in Java "

If your introductory course was in a language like Scheme, Javascript, or another scripting language, you might still be OK, though it will be a bit more challenging because of the difference between Java and these languages. You might try this course, but keep an open mind to falling back to the Duke specialization if you find you are in over your head.
