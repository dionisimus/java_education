Is this course right for me?

This course is designated as Intermediate Level, with "some experience required." But you're probably wondering just what experience, exactly, is required. In this reading and the quiz that follows, we will try to help you determine whether or not you have the correct background for this course and this specialization.

If you're coming to this course as the third course in the Object Oriented Programming in Java specialization after completing the first two courses with our colleagues from Duke, you're almost certainly ready for this course. But just to make sure, make sure everything in the list below looks familiar to you.

We assume that you have taken and passed a college-level introductory Java programming course designed for CS majors , or its online equivalent. Topics in these courses typically include at least the following topics:

  • Variables and types

  • Basic data types: Strings, ints, doubles, etc.

  • Loops (for, while, nested loops)

  • Calling methods

  • Writing methods (with and without return types, with and without parameters)

  • Using objects and possibly designing them

  • 1D and 2D arrays

  • Basic input and output (System.out.println)

  • Compiling and running Java code

One example of such a course is UC San Diego's CSE 8A. If the content on that page looks familiar to you, you're in good shape.

Here at Coursera, we've designed this course to fit into two places: (1) as the first course in the Object Oriented Java Programming: Data Structures and Beyond specialization and (2) as the third course in the Object Oriented Programming in Java specialization. The Object Oriented Java Programming: Data Structures and Beyond specialization was designed as a follow on to the " Java Programming: An Introduction to Software " specialization, offered by Duke University. So if you are brand new to programming you might want to start there first. Or you can start with the first course in the Object Oriented Programming in Java specialization (which is the second course in the Java Programming: An Introduction to Software specialization) if you've already got a little programming experience.

What if I have taken an introductory programming course, but it wasn't in Java?

If you have taken a course that covers all or most of the above topics, but it was in a language like C or C++, you will probably do fine in our specialization. We recommend that you review Java syntax and some details of how Java works with objects using some basic online resources (e.g. http://www.tutorialspoint.com/java/java_basic_syntax.htm ).

If your introductory course was in a language like Scheme, Python or Javascript or another scripting language, you might still be OK, though it will be a bit more challenging because of the difference between Java and these languages. You might try this course, but keep an open mind to falling back to the Duke specialization if you find you are in over your head.
