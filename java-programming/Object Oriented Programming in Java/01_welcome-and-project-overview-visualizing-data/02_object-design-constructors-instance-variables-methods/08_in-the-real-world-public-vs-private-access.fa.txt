[موسیقی]. من به عنوان مهندس نرم افزار، هر روز از کلمات عمومی و خصوصی استفاده می کنم. گرچه ممکن است بی اهمیت به نظر برسد ، بسیار مهم است که از این کلمات درست و بجا استفاده کنیم. شما همه چیز را خصوصی می کنید و در نهایت با کدی مواجه می شوید که نمیتواند با هیچ چیز دیگری تعامل کند شما همه چیز را عمومی کنید و در نهایت با کدی مواجه می شوید که بسیار آسیب پذیر و حتی بی فایده است.
