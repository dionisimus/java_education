2. If you know that a problem is considered NP Hard, that means:

  • You can be sure there is no polynomial time - O(n^k) - solution. (incorrect)

Remember that no one knows whether there are polynomial time solutions to some of the problems that are NP Hard. If P = NP, then there will be a polynomial time solution to some NP Hard problems (namely, those in NP).

  • You can be sure that the problem is at least as hard as a set of problems where there is no known polynomial - O(n^k) - solution. (correct)

Great response! These problems are either in NP or harder than those in NP.

  • You can be sure there is a polynomial time - O(n^k) - solution. (incorrect)

In fact, no one knows whether P = NP. But even if so, there will still be NP Hard problems (which are not in NP) that don't have a polynomial time solution.

5. A Hamiltonian graph is one where there's some path through the graph which visits every vertex exactly once. An Eulerian graph is one where there's some path through the graph which traverses every edge exactly once.

Which of the following is true about Hamiltonian and Eulerian graphs?

(Select all that apply.)

  • Every Hamiltonian graph is Eulerian, because every Hamiltonian path uses each edge at most once. (incorrect)

A Hamiltonian path just has to visit each vertex exactly once. It might not use all of the edges.

  • Each graph is either Hamiltonian or Eulerian. (incorrect)

A graph might be both Hamiltonian and Eulerian, or neither Hamiltonian nor Eulerian.

  • There can be a graph that is Hamiltonian but not Eulerian. (correct)

An example of such a graph is a fully connected graph with 4 nodes.
