Is this course right for me?

This course is designated as Intermediate Level, with "some experience required." But you're probably wondering just what experience, exactly, is required. In this reading and the quiz that follows, we will try to help you determine whether or not you have the correct background for this course and this specialization.

We assume that you have taken and passed a college-level introductory Object Oriented Java programming course designed for CS majors ("CS1") and completed at least part of an introductory data structures course ("CS2") , or their online equivalents. Topics in these courses typically include at least the following topics:

  • Variables and types

  • Basic data types: Strings, ints, doubles, etc.

  • Loops (for, while, nested loops)

  • Calling methods

  • Writing methods (with and without return types, with and without parameters)

  • Using objects and possibly designing them

  • 1D and 2D arrays

  • Basic input and output (System.out.println)

  • Compiling and running Java code

  • Creating and designing classes

  • Variable Scope

  • Inheritance and Polymorphism

  • Abstract classes and Interfaces

  • Searching and sorting arrays

  • Running time analysis using big-O notation

  • Arrays, Linked Lists and trees (including binary search trees): both use and implementation

One example of of a CS1 course is UC San Diego's CSE 11. The relevant CS2 course at UC San Diego is CSE12 , which we would expect you to know at least some of the material for. If the content in the course schedules on these pages looks familiar to you, you're in good shape. Here at Coursera, we've designed this course to be the third in our intermediate programming specialization, following our first two courses: " Object Oriented Programming in Java " and " Data Structures: Measuring and Optimizing Performance ". If you have some background in Java but are unfamiliar with some of the topics above (like Inheritance), we recommend you start at either the first or second course, whichever is more appropriate. Each previous course has a similar "Is this course right for me?" guide to help you decide.

If you are brand new to programming, you may want to start with the introductory Java specialization " Java Programming: An Introduction to Software " offered by Duke University.

What if I have taken "CS1" (introductory programming) and "CS2" (basic data structures) courses, but they weren't in Java?

If you have taken courses that cover all or most of the above topics, but it was in a language like C or C++, you will probably do fine in our specialization. We recommend that you review Java syntax and some details of how Java works with objects using some basic online resources (e.g. http://www.tutorialspoint.com/java/java_basic_syntax.htm ). You may benefit from reviewing information in this specialization's first course: " Object Oriented Programming in Java "

If your introductory courses were in a language like Scheme, Python or Javascript or another scripting language, you might still be OK, though it will be a bit more challenging because of the difference between Java and these languages. You might try this course, but keep an open mind to falling back to the Duke specialization if you find you are in over your head.
