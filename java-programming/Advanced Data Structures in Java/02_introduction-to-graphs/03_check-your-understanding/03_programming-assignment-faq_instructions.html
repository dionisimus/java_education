Q: The score the local grader gives me is different from the score I get when I submit the assignment. What's going on?

A: Make sure you are running the correct graders, and all of the graders, for this assignment . The most common reason for this error is that people fail to run one of the two graders--e.g. they run GraphGrader but not DegreeGrader. This assignment has two parts and two different graders: GraphGrader.java and DegreeGrader.java.

If that doesn't fix your problem, try posting in the discussion forum to see if anyone else has encountered something similar.

If none of these fix your problem, submit a bug report and we will look into it.

Q: I am getting a Grader Failed message. Is this something wrong with my code, or your graders?

A: Chances are this is because your code is doing something very unexpected, that we were not able to anticipate with our graders. This can happen if you submit the wrong files (e.g. class files instead of .java files). Check to make sure you've submitted the correct files exactly as described in the assignment instructions.

If you don't see anything that you are doing wrong, post in the discussion forum for help. As a last resort you can submit a bug report.

Q: How can I test degreeSequence? Graph.java is an abstract class so I can't instantiate it.

A: You can test degreeSequence by creating an instance of either GraphAdjMatrix or GraphAdjList, which are both non-abstract. Then, design a graph structure you want to test on and add the vertices and edges to your graph using addVertex and addEdge. Finally, call degreeSequence on your graph object and print out the result to check it. Alternatively, after adding your Vertices and Edges you can just print the graph, which will call degreeSequence for you and print out the degree sequence for the graph you created.

You can use this process to test the other methods you have to write for this assignment as well.

Of course, you're welcome to write JUnit tests as well if you get inspired to do so. :)
